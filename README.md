# animated-page

This is a React Dashboard App.

## Get it up and running

- Clone this repository.

``` bash
$ https://github.com/tubagusivan/animated-page.git
```

- Switch to the cloned directory.

``` bash
$ cd animated-page
```

- Start the stack.

``` bash
$ npm install
```

``` bash
$ npm start
```

- Visit `localhost:3000` in browser.
